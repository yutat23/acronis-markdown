# acronis-markdown

AcronisクラウドストレージでMarkdownファイルをWeb上でプレビューするChrome拡張機能

<img alt="image" src="https://github.com/user-attachments/assets/03e3a112-d1b8-4b6c-bba1-03cc5e914248" />

## できること

- Markdownファイルのプレビュー/Raw表示
- Web操作のみの編集・アップロード

## インストール

1. Chromeで `chrome://extensions/` を開く
2. 「デベロッパーモード」をONにする
3. 「パッケージ化されていない拡張機能を読み込む」でこのフォルダを選択
